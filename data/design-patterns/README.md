# Design Patterns Memory

This folder stores learned UI design patterns, fonts, and interaction behaviors.

When you generate a website from a URL or describe designs, the system memorizes:
- **Interactive patterns**: Click-to-redirect, dropdown-on-hover, carousel arrows, etc.
- **Font patterns**: Heading/body/logo fonts, weights, letter-spacing
- **Layout patterns**: Hero styles, nav structure, section types
- **Latest trends**: Color palettes, typography choices, common layouts

Patterns are loaded when generating new sites so the AI applies learned trends.
